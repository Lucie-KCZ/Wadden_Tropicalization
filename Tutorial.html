Tropicalization in the Wadden Sea

Lucie Kuczynski

2025-01-22

Introduction

This tutorial explains how to analyze tropicalization in marine communities using three scripts that work together to: 1. Extract temperature data for species occurrences 2. Calculate Species Temperature Index (STI) 3. Analyze Community Temperature Index (CTI) and assess tropicalization processes

Prerequisites

Before starting, ensure you have: - R installed with required packages - Your community data in wide format (species as columns) - Access to temperature data (MODIS SST rasters)

Required R packages:

install.packages(c("terra", "nlme", "MuMIn", "ggeffects", "scales", "ggplot2"))

Directory Structure

⚠️ Important: Your directory structure should look like this:

project/
├── data/
│   └── your_community_data.csv
└── scripts/
    ├── extract_temperature.R
    ├── compute_STI.R
    └── compute_CTI.R

⚠️ Note: The temperature data are/can be in a different folder.

Step 1: Extract Temperature Data

Purpose

Extract temperature values for each species occurrence from MODIS SST raster data.

Critical Setup Steps

  1. Prepare your community data CSV with these required columns:
    • year
    • lat (latitude)
    • long (longitude)
    • Species abundance columns
  2. Set up file paths:
data_name <- "mzb"
community_file <- paste0("../data/", data_name, "_wide.csv")
data_dir <- "/Users/lucie/Documents/Work/Data/TerraModis/data/processed/AquaMODIS"
avg_file <- "/Users/lucie/Documents/Work/Data/TerraModis/data/processed/average_sst.tif"
output_csv <- paste0("../data/", data_name, "_temperatures.csv")

community <- read.csv(community_file)[, -1]
year_range <- range(community$year)

⚠️ Warning: The most common errors come from incorrect file paths. Double-check your paths!

Running the Temperature Extraction

source("extract_temperatures.R")

Step 2: Calculate Species Temperature Index (STI)

Purpose

Calculate the average temperature experienced by each species across all its occurrences.

Setup

  1. Ensure your temperature-updated community file exists (from Step 1)
  2. Set your data name and directories:
data_name <- "mzb"
input_dir <- "../data"
output_dir <- "../data"

Running STI Calculation

source("compute_STI.R")
results <- compute_STI(
  data_name = data_name,
  input_dir = input_dir,
  output_dir = output_dir
)

Step 3: Analyze Community Temperature Index (CTI)

Purpose

Calculate community-level temperature indices and analyze tropicalization processes.

Setup and Data Loading

input_dir <- "../data"
output_dir <- "../data"

# Load data

                               paste0(data_name, "_wide.csv")), 
                     stringsAsFactors = FALSE)[, -1]
computed_STI <- read.csv(file.path(input_dir, 
                                  paste0(data_name, "_STI.csv")))

# Ensure correct column names
colnames(community)[2:3] <- c("Station", "Year")

# Load the functions
source("compute_CTI.R")

Common Issues and Solutions

  1. File Not Found Errors
    • Double-check all file paths
    • Use absolute paths for MODIS data
    • Ensure consistent naming conventions
  2. Missing Columns
    • Verify column names match exactly
    • Check for case sensitivity
    • Look for hidden spaces in column names
  3. Data Format Issues
    • Ensure numeric columns are actually numeric
    • Check for NA values in critical columns
    • Verify date formats are consistent

Understanding the Outputs

CTI Plot

  • Shows temporal trend in community temperature
  • Gray indicates non-significant trend
  • Red indicates significant warming trend

Process Analysis

Four possible processes are identified: 1. Tropicalization: Warm-water species increasing 2. Borealisation: Cold-water species increasing 3. Detropicalization: Warm-water species decreasing 4. Deborealisation: Cold-water species decreasing

The size of points in the species plot indicates the magnitude of temperature difference from community average.
