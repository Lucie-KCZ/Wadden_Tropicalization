Tropicalization in the Wadden Sea

Lucie Kuczynski

2025-02-11

Introduction

This tutorial explains how to analyze tropicalization in marine communities using three scripts that work together to:

  1. Extract temperature data for species occurrences
  2. Calculate Species Temperature Index (STI)
  3. Analyze Community Temperature Index (CTI) and assess tropicalization processes

Prerequisites

Before starting, ensure you have:

  • R version 4.0.0 or higher
  • Required R packages (automatically installed in setup)
  • Your community data in wide format (species as columns)
  • Access to temperature data (MODIS SST rasters)

Step 1: Extract Temperature Data

Goal

Extract temperature values for each species occurrence from MODIS SST raster data.

Setup - critical

  1. Prepare your community data CSV with these required columns:
  • SampleID
  • StationID
  • year
  • lat (latitude)
  • long (longitude)
  • Species abundance columns
  1. Set up file paths:
data_name <- "mzb" # MODIFY THE NAME OF THE DATASET

base_dir <- "/Users/lucie/Dropbox/Wadden_Tropicalization" # MODIFY THIS PATH

output_dir <- file.path(base_dir, "output")

temperature_output <- file.path(output_dir, paste0(data_name, "_temperatures.csv"))

# Import the community data

Visualize species trends

plot_species(inferred_processes$trends)

The analysis identifies four possible processes:

  1. Tropicalization: Warm-water species increasing = Indicates community shifting toward warmer-water species
  2. Borealisation: Cold-water species increasing = Indicates community shifting toward colder-water species
  3. Detropicalization: Warm-water species decreasing = Loss of warm-water species
  4. Deborealisation: Cold-water species decreasing = Loss of cold-water species

In the species plot:

  • Point size indicates magnitude of STI difference from community average
  • Colors indicate different processes
  • Position shows relationship between species thermal preference (STI) and population trend

Troubleshooting

Common Issues and Solutions

  1. File Not Found Errors
  • Verify all file paths are correct
  • Use file.path() instead of string concatenation
  • Check file permissions
  1. Missing Columns
  • Ensure column names match exactly (case-sensitive)
  • Remove any hidden characters in column names
  • Use colnames() to verify column names
  1. Data Format Issues
  • Check data types: str(community)
  • Look for NA values: colSums(is.na(community))
  • Verify numeric columns: sapply(community, class)
  1. Memory Issues
  • Close other R sessions
  • Use gc() to free memory
  • Consider processing data in chunks
